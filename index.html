Progressive Enhancement

Wat is Progressive Enhancement?

Progressive enhancement is de techniek waarbij op de eerste plaats op de content en de structuur wordt gefocust. Door progressive enhancement toe te passen, wordt ervoor gezorgd dat de website toegankelijk is voor iedereen. Zo wordt er voor gezorgd dat mensen die bijvoorbeeld een oudere versie van een browser, die geen CSS en/of JavaScript ondersteund, de hoofdfunctionaliteiten op de website alsnog kunnen gebruiken. Zo wordt allereerst alleen HTML aan de gebruiker getoond, waardoor echt alleen de hoofdfunctionaliteiten op de website beschikbaar zijn. De website is nu puur functioneel, zonder enige andere toevoegingen. Vervolgens wordt er een laagje CSS aan toegevoegd, waardoor het geheel beter bruikbaar wordt. Zo kan de lay-out duidelijker worden en het geheel aantrekkelijker wordt. Als laatste komt er een laagje JavaScript overheen. Deze laag zorgt ervoor dat het geheel prettig wordt om te gebruiken. Hierdoor wordt, volgens Aarron Walter, de gebruikerservaring per laag verbeterd.

Progressive Enhancement als metafoor

Je kunt progressive enhancement zien als een taart. We beginnen met een normaal, kaal stuk taart. Deze basis kunnen we zien als de content, welke bestaat uit semantische HTML code. Door gebruik te maken van semantische code, zorgen we ervoor dat elke gebruiker de content goed kan bereiken en gebruiken. Deze basis is direct ook het grootste en meest belangrijke stuk van de taart. Vervolgens voegen we er een laag glazuur aan toe. Deze laag verbetert het uiterlijk en de smaak van de taart. Deze toevoeging kunnen we zien als de CSS, welke ervoor zorgt dat de content op een duidelijke, fijne manier wordt weergegeven. Als laatste voegen we decoratie toe aan onze taart.
